Aide mémoire sélécteurs css 3

Les sélecteurs css

  1. * : selectionne TOUS les éléments de la page HTML
  2. .ma-jolie-class : séléctionne tous les éléments ayant le nom de class "ma-jolie-class"
  3. #mon-unique-id : séléctionne tous les éléments ayant le nom d'ID "mon-unique-id"
  4. div, aside, article : séléctionne toutes les balises <div>, <aside>, et <article>
  5. nav a : sélectionne les liens <a> imbriqués dans la balise <nav>
  6. nav a:hover : sélectionne tous les liens de la nav et ective une animation au survol de la souris

propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possible

  1. Propriétés pour le texte
    • font-family: pour définir la famille de typo dans un ordre de priorité
    • font-size: pour définir la taille de la typo, des caractères
    • line-height: interllignage, espace entre la lignes
    • text-align: alignement du texte, à gauche, à droite ou au centre
    • text-decoration : pour "décorer" le texte en ajoutant une ligne en dessous
    • color: pour définir une couleur du prmiér plan d'un élément de texte : typo, de la font ou du texte
  2. Propriétés pour les éléments
    • background: pour définir un font ou arrière plan d'une couleur d'un élément html
    • height: pour définir l'hauteur de l'élément.
    • width: pour définir la largeur de l'élément : tous les deux par rapport à la taille de la fenêtre si on indique une valeur proportionnelle en % ou en em
    • left: position à partie de la gauche de la fenêtre du navigateur.
    • margin: pour définir la marge extérieur d'un élément
    • padding: pour définir la marge intèrieur d'un élémen
    • position: position de l'élément dans la pag
    • top: position par rapport au haut de la fenêtre
    • transition: temps de transition lors d'une animationul
